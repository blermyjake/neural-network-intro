We'll be teaching a dog to either sit, run, or jump

  • If you clap, he must sit down.
  • If you whistle, he must run.
  • If you clap AND whistle, he must jump.

Please Provide an action set and a learning rate.

The first element of the input will be 1 if you are clapping, and 0 otherwise. Similarly, the second element will be 1 if you are whistling, and 0 otherwise.

A very low learning rate will usually make the neural network a slow, but very good learner. On the other hand, a very high learning rate will make it learn quickly, but also very erratically.
